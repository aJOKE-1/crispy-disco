##贪吃蛇介绍

思路：
1.首先制作出贪吃蛇蛇身、墙、食物
2.考虑其移动方式，通过键盘输入控制蛇头方向，进而移动蛇身
3.考虑蛇吃食物的方式，和积分
4.考虑蛇会出现的问题，即游戏结束的原因，撞墙、撞身
5.如何将地图显示出来


具体过程：
1.初始化蛇身，地图，这里表示蛇的方法是使用二维数组，即snake[x][y];x=0时，y表蛇的横坐标，x=1时，y表蛇的纵坐标；地图也用二维数组表示。设置蛇的长度，表示积分。
2.设置获得指令函数，即得知蛇的前进方向，判断是否与原方向相同或相反。
3.设置重新获得食物指令，在生成食物时需判断食物是否与蛇身相重和，在蛇吃掉上一个食物后，用rand函数随机生成种子的方法。食物表示方法同蛇。
4.制作地图函数，用不同的符号代表蛇身、食物、墙。
5.设置蛇移动的函数，得到指令，先移动头，再依次移动蛇身，采用补位法。
6.判断食物是否会被吃掉，即是否蛇头位置与食物位置相同。
7.进行判断蛇是否会发生碰撞，通过比较蛇头的位置和墙或者蛇身的位置相重合
8.最后进行主函数，先进行初始化，展示地图，然后进入无限循环，先得出键盘输入的方向，判断，成功后移动蛇，判断其是否吃掉食物，同时判断是否发生碰撞，进行清屏，
  sleep;再重新制作地图，如此往复。如发生碰撞，则会显示游戏结束的字眼，并显示积分。
